WhatsApp Message Scheduler Using Python

This project is a Python-based automation tool that allows users to schedule and send WhatsApp messages at a specific time. The application takes the recipient's phone number, message content, and the desired sending time as input. Using the pywhatkit library, it automatically opens WhatsApp Web and sends the message at the scheduled time. This tool helps users automate reminders, notifications, greetings, and other time-sensitive messages without manual intervention.

Key Features:Accepts recipient phone number and message as input.
Allows users to schedule messages for a specific date and time.
Automatically sends WhatsApp messages using WhatsApp Web.
Simple and user-friendly command-line interface.
Includes basic error handling for invalid time formats and sending failures.

Technologies Used:

Python
PyWhatKit Library
Datetime Module
